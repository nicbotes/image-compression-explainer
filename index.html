Image Compression Algorithms

Rasterised Images

Rasterised images are made up of pixels and are resolution-dependent. Compression techniques for rasterised images can be either lossy or lossless.

Lossy Compression - JPEG

JPEG: Joint Photographic Experts Group. Lossy compression that reduces file size by discarding some data, optimised for photos.

Goal: Reduce file size with minimal visual loss, often used for digital photography and web images.

Lossless Compression - PNG

PNG: Portable Network Graphics. Lossless compression that retains image data without degradation, optimised for images with text or graphics.

Goal: Maintain image quality while reducing file size, often used for logos or images with transparency.

Vector Images

Vector images are resolution-independent and consist of geometric shapes like lines, points, and curves, making them scalable without losing quality.

SVG

Scalable Vector Graphics: An XML-based format for vector images, often used for icons, logos, and illustrations.

Goal: Achieve high scalability without loss of quality, keeping file sizes minimal for icons and web graphics.
